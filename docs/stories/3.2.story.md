# Story 3.2 — Página Lista de OPs com filtros e paginação

**Epic:** 3 — Frontend (Telas)
**Story:** 3.2
**Status:** Done
**Criado por:** @sm (River)
**Data:** 2026-06-30

---

## Story

**Como** operador, supervisor ou gerente,
**Quero** navegar pela lista de ordens de produção com filtros por data, classe e busca por número,
**Para que** eu possa localizar rapidamente uma OP específica ou visualizar o histórico de um período.

---

## Acceptance Criteria

- [ ] AC-01: A página `/ops` exibe lista paginada de OPs com colunas: Número da OP, Data, Total de Couros, Aproveitamento Médio, Classe Predominante
- [ ] AC-02: A página exibe controles de filtro: período (data início / data fim), classe predominante (select), busca por número de OP (texto)
- [ ] AC-03: Ao aplicar filtros, a lista é recarregada com os parâmetros correspondentes via `GET /ops`
- [ ] AC-04: A paginação exibe botões anterior/próximo e número da página atual
- [ ] AC-05: Clicar em uma OP navega para `/ops/{op}`
- [ ] AC-06: Durante o carregamento, exibe indicador visual de loading
- [ ] AC-07: Se a API retornar erro, exibe mensagem de erro amigável
- [ ] AC-08: A lista exibe mensagem "Nenhuma OP encontrada" quando `items` está vazio
- [ ] AC-09: Os filtros são refletidos na URL como query params (ex: `?page=2&search=12345`) para permitir compartilhamento de link
- [ ] AC-10: `npm run build` passa sem erros de TypeScript após a implementação

### Sincronização com URL (query params)

Usar `useSearchParams` do `react-router-dom` para ler e escrever filtros na URL:

```typescript
import { useSearchParams } from 'react-router-dom';

const [searchParams, setSearchParams] = useSearchParams();

const page = Number(searchParams.get('page') || '1');
const search = searchParams.get('search') || '';
const dateFrom = searchParams.get('date_from') || '';
const dateTo = searchParams.get('date_to') || '';
const classFilter = searchParams.get('class') || '';

// Ao aplicar filtro:
const handleFilter = () => {
  setSearchParams({ page: '1', search, date_from: dateFrom, date_to: dateTo, class: classFilter });
};
```

### Chamada à API com Filtros Dinâmicos

```typescript
useEffect(() => {
  const params: Record<string, string> = { page: String(page), limit: '20' };
  if (search) params.search = search;
  if (dateFrom) params.date_from = dateFrom;
  if (dateTo) params.date_to = dateTo;
  if (classFilter) params.class = classFilter;

  client.get<OpListResponse>('/ops', { params })
    .then(res => setData(res.data))
    .catch(() => setError('Erro ao carregar lista de OPs'))
    .finally(() => setLoading(false));
}, [page, search, dateFrom, dateTo, classFilter]);
```

### Componente OpTable

```typescript
// src/components/tables/OpTable.tsx
import { Link } from 'react-router-dom';
import { OpSummary } from '../../types/op';

interface Props {
  items: OpSummary[];
}

export default function OpTable({ items }: Props) {
  if (items.length === 0) return <p>Nenhuma OP encontrada.</p>;

  return (
    <table>
      <thead>
        <tr>
          <th>OP</th>
          <th>Data</th>
          <th>Couros</th>
          <th>Aproveitamento</th>
          <th>Classe Predominante</th>
        </tr>
      </thead>
      <tbody>
        {items.map(op => (
          <tr key={op.op}>
            <td><Link to={`/ops/${op.op}`}>{op.op}</Link></td>
            <td>{new Date(op.last_updated).toLocaleDateString('pt-BR')}</td>
            <td>{op.total_hides}</td>
            <td>{op.avg_yield?.toFixed(1)}%</td>
            <td>{op.predominant_class}</td>
          </tr>
        ))}
      </tbody>
    </table>
  );
}
```

### Componente Pagination

```typescript
// src/components/ui/Pagination.tsx
interface Props {
  page: number;
  pages: number;
  onPageChange: (page: number) => void;
}

export default function Pagination({ page, pages, onPageChange }: Props) {
  return (
    <div>
      <button disabled={page <= 1} onClick={() => onPageChange(page - 1)}>Anterior</button>
      <span>Página {page} de {pages}</span>
      <button disabled={page >= pages} onClick={() => onPageChange(page + 1)}>Próximo</button>
    </div>
  );
}
```
