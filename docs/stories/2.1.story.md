## Story

**Como** usuário do portal Lizard Web,
**Quero** acessar um endpoint de dashboard que consolide métricas do dia e da semana,
**Para que** o frontend possa exibir uma visão rápida da produção recente sem precisar montar agregações no cliente.

---

## Acceptance Criteria

- [ ] AC-01: `GET /dashboard` retorna JSON com campos `today.total_hides`, `today.avg_yield`, `week.total_hides`, `week.avg_yield`, `today_class_distribution` e `recent_ops`
- [ ] AC-02: `today` agrega registros cujo `GrdDat` está na data atual (localtime do servidor)
- [ ] AC-03: `week` agrega registros dos últimos 7 dias corridos (incluindo hoje)
- [ ] AC-04: `today_class_distribution` retorna lista de `{class, count, pct}` ordenada por `count` decrescente
- [ ] AC-05: `recent_ops` retorna as últimas 5 OPs distintas processadas (por `GrdDat` decrescente) com campos `op`, `date`, `total_hides`
- [ ] AC-06: Toda query usa parameterized statements — sem concatenação de strings em SQL
- [ ] AC-07: Endpoint retorna 200 mesmo quando não há dados (retorna zeros/listas vazias, nunca 500)
- [ ] AC-08: `lizard-web/api/routers/ops.py` não importa nenhum módulo de `lizard/`

### Queries de Referência
[Source: docs/architecture/system-architecture.md#4-api-contratos-de-endpoints]

```python
# Agregação do dia (usar date() do SQLite para comparar apenas a data)
cursor.execute("""
    SELECT COUNT(*) as total, AVG(GrdUsaPer) as avg_yield
    FROM grading_results
    WHERE date(GrdDat) = date('now', 'localtime')
""")

# Distribuição de classes do dia
cursor.execute("""
    SELECT GrdCla as class,
           COUNT(*) as count,
           ROUND(COUNT(*) * 100.0 / SUM(COUNT(*)) OVER (), 1) as pct
    FROM grading_results
    WHERE date(GrdDat) = date('now', 'localtime')
    GROUP BY GrdCla
    ORDER BY count DESC
""")

# Semana (últimos 7 dias)
cursor.execute("""
    SELECT COUNT(*) as total, AVG(GrdUsaPer) as avg_yield
    FROM grading_results
    WHERE date(GrdDat) >= date('now', '-6 days', 'localtime')
""")

# OPs recentes (últimas 5 distintas)
cursor.execute("""
    SELECT GrdOp as op,
           date(MAX(GrdDat)) as date,
           COUNT(*) as total_hides
    FROM grading_results
    GROUP BY GrdOp
    ORDER BY MAX(GrdDat) DESC
    LIMIT 5
""")
```

class DashboardPeriod(BaseModel):
