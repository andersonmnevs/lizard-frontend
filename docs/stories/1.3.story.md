## Dev Notes

### Estrutura de Rotas
[Source: docs/architecture/system-architecture.md#3-estrutura-de-pastas e #4-api-contratos-de-endpoints]

```tsx
// App.tsx
import { BrowserRouter, Routes, Route } from 'react-router-dom'
import Layout from './components/ui/Layout'
import Dashboard from './pages/Dashboard'
import OpList from './pages/OpList'
import OpDetail from './pages/OpDetail'
import HideDetail from './pages/HideDetail'

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Layout />}>
          <Route index element={<Dashboard />} />
          <Route path="ops" element={<OpList />} />
          <Route path="ops/:id" element={<OpDetail />} />
          <Route path="ops/:opId/hides/:hideId" element={<HideDetail />} />
        </Route>
      </Routes>
    </BrowserRouter>
  )
}
```

### Cliente Axios
[Source: docs/architecture/system-architecture.md#2-stack-tecnológica]

```typescript
// src/api/client.ts
import axios from 'axios'

const client = axios.create({
  baseURL: import.meta.env.VITE_API_URL || 'http://localhost:8000',
  headers: { 'Content-Type': 'application/json' }
})

export default client
```

Em desenvolvimento, o proxy do Vite redireciona `/api` para `http://localhost:8000`. Em produção, a API e o frontend rodam na mesma origem (FastAPI serve o `dist/`).

### Interfaces TypeScript
[Source: docs/architecture/system-architecture.md#4-api-contratos-de-endpoints]

```typescript
// src/types/op.ts
export interface OpSummary {
  op: string

export interface OpDetail {
  op: string
  total_hides: number
  avg_area: number | null
  avg_yield: number | null
  class_distribution: ClassDistribution[]

export interface ClassDistribution {
  class: string

export interface Hide {
  id: number
  hide_num: string
  op: string
  processed_at: string
  class: string
  yield_pct: number | null
  area: number | null
  image_available: boolean
  prev_hide_id: number | null
  next_hide_id: number | null
}
```

### Layout Base
O header deve conter:
- Nome da aplicação: "Lizard Web"
- Link para Dashboard (`/`)
- Link para Lista de OPs (`/ops`)

Usar `<Outlet />` do `react-router-dom` para renderizar a página filha dentro do Layout.

### Variáveis de Ambiente Vite
Criar `lizard-web/frontend/.env.example`:
```
VITE_API_URL=http://localhost:8000
```

### Build de Produção
`npm run build` gera `frontend/dist/`. Na Story 1.2 (ou 2.x), o FastAPI será configurado para servir este diretório como arquivos estáticos.
