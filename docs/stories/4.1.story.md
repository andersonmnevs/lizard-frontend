## Story

**Como** supervisor ou gerente,
**Quero** exportar um relatório PDF de uma OP com tabela de couros e resumo agregado,
**Para que** eu possa arquivar ou compartilhar o resultado da produção fora do portal.

## Acceptance Criteria

- [x] AC-01: `GET /ops/{op_id}/export/pdf` gera e retorna um arquivo PDF válido
- [x] AC-02: O PDF contém cabeçalho com número da OP e período (data mais antiga a mais recente dos couros)
- [x] AC-03: O PDF contém tabela com todos os couros da OP: número do couro, data/hora, classe, aproveitamento (%), área (m²)
- [x] AC-04: O PDF contém seção de resumo: total de couros, aproveitamento médio, área média, distribuição de classes
- [x] AC-05: O response tem header `Content-Disposition: attachment; filename="op-{op_id}.pdf"` e `Content-Type: application/pdf`
- [x] AC-06: Retorna 404 com `{"detail": "OP não encontrada"}` se `op_id` não existir
- [x] AC-07: Sem imagens no PDF (conforme PRD seção 3.5)
- [x] AC-08: Toda query ao SQLite usa parameterized statements
- [x] AC-09: `lizard-web/api/routers/exports.py` não importa nenhum módulo de `lizard/`
- [x] AC-10: O PDF é gerado sob demanda — sem pré-geração ou cache de arquivo

### Implementação com WeasyPrint
[Source: docs/architecture/system-architecture.md#2-stack-tecnológica]

WeasyPrint converte HTML+CSS em PDF. Fluxo:

```python
# lizard-web/api/routers/exports.py
import io
from fastapi import APIRouter, HTTPException
from fastapi.responses import StreamingResponse
from weasyprint import HTML
from api.db import get_db_connection

router = APIRouter()

@router.get("/ops/{op_id}/export/pdf")
def export_op_pdf(op_id: str):
    with get_db_connection() as conn:
        # 1. Verificar existência da OP
        cursor = conn.cursor()
        cursor.execute("SELECT COUNT(*) FROM grading_results WHERE GrdOp = ?", (op_id,))
        if cursor.fetchone()[0] == 0:
            raise HTTPException(status_code=404, detail="OP não encontrada")

        # 2. Dados agregados
        cursor.execute("""
            SELECT
                COUNT(*) as total_hides,
                ROUND(AVG(GrdAre), 2) as avg_area,
                ROUND(AVG(GrdUsaPer), 2) as avg_yield,
                MIN(date(GrdDat)) as date_from,
                MAX(date(GrdDat)) as date_to
            FROM grading_results WHERE GrdOp = ?
        """, (op_id,))
        summary = cursor.fetchone()

        # 3. Distribuição de classes
        cursor.execute("""
            SELECT GrdCla as class, COUNT(*) as count
            FROM grading_results WHERE GrdOp = ?
            GROUP BY GrdCla ORDER BY count DESC
        """, (op_id,))
        classes = cursor.fetchall()

        # 4. Lista de couros (todos, sem paginação para PDF)
        cursor.execute("""
            SELECT GrdHidNum, GrdDat, GrdCla, GrdUsaPer, GrdAre
            FROM grading_results WHERE GrdOp = ?
            ORDER BY GrdHidNum ASC
        """, (op_id,))
        hides = cursor.fetchall()

    # 5. Gerar HTML
    html_content = build_pdf_html(op_id, summary, classes, hides)

    # 6. Converter para PDF
    pdf_bytes = HTML(string=html_content).write_pdf()

    # 7. Retornar como streaming
    return StreamingResponse(
        io.BytesIO(pdf_bytes),
        media_type="application/pdf",
        headers={"Content-Disposition": f'attachment; filename="op-{op_id}.pdf"'}
    )
```

### Template HTML para o PDF

```python
def build_pdf_html(op_id, summary, classes, hides) -> str:
    total = summary["total_hides"]
    class_rows = "".join(
        f"<tr><td>{c['class']}</td><td>{c['count']}</td>"
        f"<td>{round(c['count']/total*100, 1)}%</td></tr>"
        for c in classes
    )
    hide_rows = "".join(
        f"<tr><td>{h['GrdHidNum']}</td><td>{h['GrdDat']}</td>"
        f"<td>{h['GrdCla']}</td><td>{h['GrdUsaPer'] or '—'}%</td>"
        f"<td>{h['GrdAre'] or '—'}</td></tr>"
        for h in hides
    )
    return f"""
    <!DOCTYPE html>
    <html>
    <head>
      <meta charset="utf-8">
      <style>
        body {{ font-family: Arial, sans-serif; font-size: 12px; margin: 2cm; }}
        h1 {{ font-size: 18px; }}
        h2 {{ font-size: 14px; margin-top: 1.5em; }}
        table {{ width: 100%; border-collapse: collapse; margin-top: 0.5em; }}
        th, td {{ border: 1px solid #ccc; padding: 4px 8px; text-align: left; }}
        th {{ background: #f0f0f0; }}
        .header {{ margin-bottom: 1em; }}
      </style>
    </head>
    <body>
      <div class="header">
        <h1>Relatório da OP {op_id}</h1>
        <p>Período: {summary['date_from']} a {summary['date_to']}</p>
        <p>Emitido por: Lizard Web — Viposa SA</p>
      </div>

      <h2>Resumo</h2>
      <table>
        <tr><th>Total de Couros</th><td>{summary['total_hides']}</td></tr>
        <tr><th>Aproveitamento Médio</th><td>{summary['avg_yield'] or '—'}%</td></tr>
        <tr><th>Área Média</th><td>{summary['avg_area'] or '—'} m²</td></tr>
      </table>

      <h2>Distribuição de Classes</h2>
      <table>
        <tr><th>Classe</th><th>Couros</th><th>%</th></tr>
        {class_rows}
      </table>

      <h2>Couros</h2>
      <table>
        <tr><th>Nº Couro</th><th>Data/Hora</th><th>Classe</th><th>Aproveit.</th><th>Área (m²)</th></tr>
        {hide_rows}
      </table>
    </body>
    </html>
    """
```
