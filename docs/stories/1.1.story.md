### Stack Tecnológica
[Source: docs/architecture/system-architecture.md#2-stack-tecnológica]

- **Backend:** Python 3.10+ / FastAPI 0.110+ / Uvicorn
- **Frontend:** React 18 / Vite 5 / TypeScript
- **Dependências backend:** `fastapi`, `uvicorn[standard]`, `python-multipart`, `weasyprint`, `openpyxl`, `pydantic`
- **Dependências frontend:** `react`, `react-dom`, `react-router-dom`, `axios`, `recharts`
- **Dev deps frontend:** `@types/react`, `@types/react-dom`, `typescript`, `@vitejs/plugin-react`

### Configuração do Proxy Vite
O proxy deve redirecionar `/api` → `http://localhost:8000` para desenvolvimento local:

```typescript
// vite.config.ts
export default defineConfig({
  plugins: [react()],
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8000',
        changeOrigin: true,
        rewrite: (path) => path.replace(/^\/api/, '')
      }
    }
  }
})
```

### Princípio de Isolamento
[Source: docs/architecture/system-architecture.md#1-visão-geral]

`lizard-web/` é completamente independente de `lizard/`. Nenhum arquivo dentro de `lizard-web/` pode importar módulos de `lizard/`. Os dois projetos compartilham apenas o sistema de arquivos (SQLite e imagens em disco).

### Arquivos Placeholder
Para que a estrutura de pastas seja commitável, criar arquivos `.gitkeep` em diretórios vazios, ou criar os arquivos Python/TSX com conteúdo mínimo (apenas imports básicos e `pass`/exportação vazia).
