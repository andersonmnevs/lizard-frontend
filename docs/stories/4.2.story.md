## Story

**Como** supervisor ou gerente,
**Quero** exportar os dados de uma OP em formato Excel com abas de couros e resumo,
**Para que** eu possa analisar os dados em planilha ou integrar com outros sistemas.

---

## Acceptance Criteria

- [x] AC-01: `GET /ops/{op_id}/export/xlsx` gera e retorna um arquivo Excel válido (`.xlsx`)
- [x] AC-02: O arquivo contém aba "Couros" com uma linha por couro: número do couro, data/hora, classe, aproveitamento (%), área (m²)
- [x] AC-03: O arquivo contém aba "Resumo" com métricas agregadas: total de couros, aproveitamento médio, área média, distribuição de classes
- [x] AC-04: O response tem header `Content-Disposition: attachment; filename="op-{op_id}.xlsx"` e `Content-Type: application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`
- [x] AC-05: Retorna 404 com `{"detail": "OP não encontrada"}` se `op_id` não existir
- [x] AC-06: Sem imagens no Excel (conforme PRD seção 3.5)
- [x] AC-07: Toda query ao SQLite usa parameterized statements
- [x] AC-08: `lizard-web/api/routers/exports.py` não importa nenhum módulo de `lizard/`
- [x] AC-09: O Excel é gerado sob demanda — sem pré-geração ou cache de arquivo
- [x] AC-10: Cabeçalhos das colunas na aba "Couros" estão em português e em negrito

### Implementação com openpyxl
[Source: docs/architecture/system-architecture.md#2-stack-tecnológica]

```python
# lizard-web/api/routers/exports.py — adicionar ao arquivo existente
import io
import openpyxl
from openpyxl.styles import Font
from fastapi.responses import StreamingResponse

@router.get("/ops/{op_id}/export/xlsx")
def export_op_xlsx(op_id: str):
    with get_db_connection() as conn:
        cursor = conn.cursor()

        # 1. Verificar existência da OP
        cursor.execute("SELECT COUNT(*) FROM grading_results WHERE GrdOp = ?", (op_id,))
        if cursor.fetchone()[0] == 0:
            raise HTTPException(status_code=404, detail="OP não encontrada")

        # 2. Dados agregados para aba Resumo
        cursor.execute("""
            SELECT
                COUNT(*) as total_hides,
                ROUND(AVG(GrdAre), 2) as avg_area,
                ROUND(AVG(GrdUsaPer), 2) as avg_yield,
                MIN(date(GrdDat)) as date_from,
                MAX(date(GrdDat)) as date_to
            FROM grading_results WHERE GrdOp = ?
        """, (op_id,))
        summary = cursor.fetchone()

        # 3. Distribuição de classes
        cursor.execute("""
            SELECT GrdCla as class, COUNT(*) as count
            FROM grading_results WHERE GrdOp = ?
            GROUP BY GrdCla ORDER BY count DESC
        """, (op_id,))
        classes = cursor.fetchall()

        # 4. Lista completa de couros para aba Couros
        cursor.execute("""
            SELECT GrdHidNum, GrdDat, GrdCla, GrdUsaPer, GrdAre
            FROM grading_results WHERE GrdOp = ?
            ORDER BY GrdHidNum ASC
        """, (op_id,))
        hides = cursor.fetchall()

    # 5. Criar workbook
    wb = openpyxl.Workbook()

    # --- Aba Couros ---
    ws_hides = wb.active
    ws_hides.title = "Couros"
    headers = ["Nº Couro", "Data/Hora", "Classe", "Aproveitamento (%)", "Área (m²)"]
    ws_hides.append(headers)
    # Negrito nos cabeçalhos
    for cell in ws_hides[1]:
        cell.font = Font(bold=True)
    for hide in hides:
        ws_hides.append([
            hide["GrdHidNum"],
            hide["GrdDat"],
            hide["GrdCla"],
            hide["GrdUsaPer"],
            hide["GrdAre"],
        ])

    # --- Aba Resumo ---
    ws_summary = wb.create_sheet("Resumo")
    total = summary["total_hides"]
    ws_summary.append(["OP", op_id])
    ws_summary.append(["Período", f"{summary['date_from']} a {summary['date_to']}"])
    ws_summary.append([])
    ws_summary.append(["Total de Couros", total])
    ws_summary.append(["Aproveitamento Médio (%)", summary["avg_yield"]])
    ws_summary.append(["Área Média (m²)", summary["avg_area"]])
    ws_summary.append([])
    ws_summary.append(["Distribuição de Classes"])
    ws_summary.append(["Classe", "Couros", "%"])
    for c in classes:
        pct = round(c["count"] / total * 100, 1) if total > 0 else 0
        ws_summary.append([c["class"], c["count"], pct])

    # 6. Serializar para bytes
    buffer = io.BytesIO()
    wb.save(buffer)
    buffer.seek(0)

    return StreamingResponse(
        buffer,
        media_type="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet",
        headers={"Content-Disposition": f'attachment; filename="op-{op_id}.xlsx"'}
    )
```

### Segurança
[Source: docs/architecture/system-architecture.md#6-segurança]

- Toda query usa `?` com parâmetros — nunca f-string em SQL
- `op_id` validado por existência antes de gerar arquivo
- openpyxl não executa macros nem código externo
